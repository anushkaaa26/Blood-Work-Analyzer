## 📌 Overview

Understanding blood test reports can be overwhelming — packed with medical jargon, reference ranges, and biomarker abbreviations that most people have never seen before.

**Blood Work Analyzer** bridges that gap. It uses **Google Gemini 2.5 Flash** to transform raw blood work data into easy-to-read health insights, flag abnormal values, and generate a **personalized Indian diet plan** — all within a clean Streamlit interface.

> 💡 **Who is this for?** Anyone who wants to better understand their blood report without waiting for a doctor's appointment or spending hours Googling medical terms.

## 🚀 Getting Started

### Prerequisites

- Python 3.9+
- A [Google Gemini API Key](https://aistudio.google.com/app/apikey)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/blood-work-analyzer.git
cd blood-work-analyzer

# 2. Install dependencies
pip install -r requirements.txt

# 3. Set up your API key
export GOOGLE_API_KEY="your_gemini_api_key_here"

# 4. Run the Streamlit app
streamlit run streamlit_app/app.py
```

> 🌐 The app will open at `https://blood-work-analyzer-zymu8945cr5keflwt6qjct.streamlit.app/`
