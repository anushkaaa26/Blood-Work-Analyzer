# 🩸 Blood Work Analyzer

An AI-powered healthcare application that analyzes blood test reports, generates personalized health insights, summarizes key findings, and recommends an Indian diet plan using **Google Gemini 2.5 Flash**.

## 🚀 Live Demo

Deployed on Streamlit for easy access and interactive health analysis.

## 📌 Overview

Understanding blood test reports can be challenging for most people due to medical terminology and complex biomarkers. This project leverages **Google Gemini 2.5 Flash** to transform raw blood work data into actionable health insights.

The application analyzes blood parameters, identifies potential health concerns, generates an easy-to-understand health summary, and recommends a personalized Indian diet plan based on the user's blood report.

## ✨ Features

- Upload and analyze blood test reports
- AI-powered health assessment using Gemini 2.5 Flash
- Identification of abnormal biomarkers
- Simplified explanations of blood parameters
- Personalized health summary
- Indian diet recommendations tailored to report findings
- Actionable lifestyle suggestions
- Interactive Streamlit web application
- Fast and user-friendly interface

## 🔄 Workflow

1. User provides blood work data.
2. Blood biomarkers are extracted and analyzed.
3. Gemini 2.5 Flash evaluates the report.
4. Health risks and observations are identified.
5. A comprehensive health summary is generated.
6. A personalized Indian diet plan is recommended.
7. The results are displayed through the Streamlit interface.
